Create navigational links within a web site

There are several techiques to navigate within a web site. The semantic tag for navigation is the <nav> tag. You can navigate through your web site by using relative addressing. For examples we will use the unordered list tag (<ul>)

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)

Navigational menu are placed in your <header> semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag like other semantic tags should have a <hx> tag. However this header is optional. If you omit a header tag you will receive a document outliner error: Untitled Section. Until you are shown how to avoid this error message, it will be acceptable on your document outliner.
